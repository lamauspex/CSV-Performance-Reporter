## [1.0.0] - 2024-11-19

### Added
- Базовый функционал для анализа CSV файлов с данными сотрудников
- Поддержка двух типов отчетов:
  - `performance` - анализ эффективности сотрудников по позициям
  - `skills` - анализ навыков сотрудников
- CLI интерфейс с поддержкой:
  - Обработки множественных файлов (`--files`)
  - Обработки целых папок (`--folder`)
  - Выбора типа отчета (`--report`)
- Красивое табличное форматирование вывода с использованием Unicode символов
- Валидация входных данных:
  - Проверка существования файлов и папок
  - Валидация структуры CSV (обязательные колонки)
  - Проверка допустимых значений для поля `performance` (0-5)
- Обработка ошибок с информативными сообщениями
- Модульная архитектура с использованием паттернов:
  - Strategy для различных типов отчетов
  - Adapter для интеграции компонентов
  - Factory в ReportGenerator
- Конфигурационная система с поддержкой .env файлов
- Полное тестовое покрытие с использованием pytest
- Поддержка анализа следующих полей сотрудников:
  - `name` - имя сотрудника
  - `position` - должность
  - `completed_tasks` - количество выполненных задач
  - `performance` - оценка эффективности (0-5)
  - `skills` - навыки (через запятую)
  - `team` - команда
  - `experience_years` - опыт работы в годах

### Technical Details
- Python 3.x с типизацией
- Основные зависимости: tabulate (для таблиц), pytest (для тестов)
- Поддержка Python 3.7+
- Протестировано на больших CSV файлах
